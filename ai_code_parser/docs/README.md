# AI Code Parser

This project is designed to parse through the code of 2-4 GitHub repos and provide instructions on how to make them cohesive together. The user will provide context on what they want out of both functionally to help navigate.

## Installation

1. Clone the repository
2. Install the dependencies with `pip install -r requirements.txt`

## Usage

1. Place the repositories you want to parse in the `data/repos` directory.
2. Run `python src/main.py`

## Dependencies

- Python
- ChatGPT3
