
# AI Agent for WordPress News Outlet

This project is an AI agent designed to automate various tasks for a WordPress news outlet. The agent is capable of content curation, SEO optimization, user interaction, automation of publishing and promotion, and editing and reviewing AI-generated content.

## Project Structure

- `src/`: Contains the source code for the AI agent.
- `tests/`: Contains the test scripts for the AI agent.
- `models/`: Contains the AI model used by the agent.
- `data/`: Contains data used and generated by the agent.
- `config/`: Contains configuration files, such as the WordPress API config.
- `logs/`: Contains log files for the agent's operations.

## Setup and Usage

1. Ensure you have Python installed on your system.
2. Install the required packages: `pip install -r requirements.txt`
3. Configure the WordPress API settings in `config/wordpress_api_config.json`.
4. Run the main script: `python src/main.py`

## Testing

To run the tests, use the following command: `python -m unittest discover tests`

## Logging

The agent logs its operations in `logs/ai_agent.log`. Check this file for detailed information about the agent's activities.
